# parachute

> cli to eject a node module into your file system

## Installation

```
npm install --global parachute
```

## Usage

```
$ parachute --help

Usage
  $ parachute [package name]

Examples
  $ parachute react-table
  //=> Eject contents of react-table in current directory
```


## Development
